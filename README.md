# Promix Accounting

A modern redesign of seamless accounting software for SMEs (Small and Medium Enterprises) in Nigeria.

## Overview

This project is a complete UI/UX revamp of the original [promixaccounting.com](https://promixaccounting.com), featuring a modern, mobile-first design with enhanced user experience and accessibility.

**Live Demo**: [https://promixaccounting.vercel.app/](https://promixaccounting.vercel.app/)

## Features

### Core Functionality
- **Seamless Accounting Management**: Streamlined account management system
- **SME-Focused Design**: Tailored specifically for Nigerian small and medium enterprises  
- **Mobile-First Approach**: Responsive design ensuring accessibility across all devices
- **Real-time Synchronization**: Instant updates across all connected devices
- **Bank-level Security**: 256-bit encryption with SOC 2 compliance
- **Global Access**: Work from anywhere, anytime with cloud-based architecture

### Key Improvements Over Original
- Modern, intuitive user interface
- Enhanced mobile responsiveness
- Improved navigation and user flow
- Better visual hierarchy and typography
- Optimized performance and loading speeds

## Tech Stack

### Frontend
- **Next.js 14** with TypeScript
- **Tailwind CSS** for modern styling
- **Responsive Design** with mobile-first approach
- **Component-based Architecture** for maintainability

### Deployment
- **Vercel** for hosting and continuous deployment
- **Optimized Performance** with automatic code splitting
- **SSL Security** with custom domain support

## Design Philosophy

- **User-Centered Design**: Prioritizing ease of use for Nigerian SMEs
- **Accessibility First**: Ensuring the platform is usable by everyone
- **Performance Optimized**: Fast loading times even on slower connections
- **Security-Focused**: Implementing financial-grade security measures

## Getting Started

```bash
# Clone the repository
git clone https://github.com/ObeeJ/promix-accounting.git
cd promix-accounting

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
